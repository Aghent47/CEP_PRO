# SPC Dashboard - Control Estadístico de Procesos

## 📋 Descripción
Aplicación web para Control Estadístico de Procesos (CEP) que permite:
- Carga de datos Excel/CSV
- Gráficos de control X-R y X-s
- Detección de 8 reglas de Nelson
- Análisis de capacidad (Cp, Cpk, PPM)
- Reportes ejecutivos PDF/CSV/TXT

## 🚀 Instalación

### Requisitos previos
- Node.js (versión 18 o superior)
- npm (versión 9 o superior)

### Pasos de instalación

```bash
# 1. Clonar o descargar el proyecto
cd spc-dashboard

# 2. Instalar dependencias
npm install

# 3. Ejecutar en modo desarrollo
npm run dev

# 4. Abrir navegador en http://localhost:5173
